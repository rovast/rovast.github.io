G2 brush 时间轴，而不是筛选数据点

一、背景

目前 G2 使用 brush-x 筛选后的是对应的点，而不是 X 轴的时间范围。在实际使用过程中，我们需要场景如下

  1. 鼠标筛选一个区域
  2. 获取这个区域的开始时间和结束时间
  3. 以第2步获取到的时间范围作为结果来重新获取数据

    • 根据 filter 进行数据筛选
  • 看下如何获取到 min value 和 max value src/interaction/action/data/range-filter.ts
